A couple of notes on the merging on 
glimmer-cism2/branches/glimmer-cism-lanl and glimmer-cism-lanl/trunk:

I have completed the first commit associated with this merge.  The 
following are several notes about what I did and did not do.

* The code I committed complies, but I do not gaurantee its 
correctness.  I tried to add comments about conflicts between the 
branches that I had to resolve.  My comments are tagged with !EIB!.
* I did not merge the tests.  However, I did modify the source for 
EISMINT/simple_forcing due to updating a subroutine call.  The tests 
need to be checked for correctness and any tests from 
glimmer-cism-lanl/trunk added as desired.
*Several features available on the glimmer-cism-lanl trunk have been 
omitted.  These include the plume work, restarts, and bwater routines.
*As much as possible I left the glide_lithot* modules alone.  The lithot 
routines differ between the two branches and should be look at more 
closely for possible merging.
* I replaced glimmer_deriv with glide_deriv.
* Several new modules were added
* There is a difference in the glimmer_version code.  Please pick the 
desired version and update all locations.

I'm sure there are lots of things I've missed in these notes, but I 
wanted to leave at least some record of I what I've done.  Hope this has 
helped.

as I said in my previous email, I would like to start with the file I/O. 
In the first instance, this would be a generic wrapper which exposes CF 
specifics such as global attributes like author, etc. Other useful 
functionality might include:
* handling projections
* applying masks
* extracting statistics
* reducing dimensionality of data datasets (or slicing them in 
interesting ways), e.g. extract data along a flowline through time.
* handling the time coordinate

I suspect that the I/O is the really tricky bit. Once we have the data, 
plotting it is relatively straight forward using matplotlib.

In terms of keeping dependencies down, I think we need
+ numpy
+ matplotlib
+ basemap

I just had a closer look at the basemap toolkit, it contains bindings to 
  proj4 and netCDF. So maybe, that's enough.

I haven't thought too much about the plotting yet, although I think we 
want to extend matplotlib axes to do CF specific plotting.

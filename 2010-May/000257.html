Hi all,
I'd like to start working on some basic visualisation python programs. 
The old stuff using GMT is beyond repair.

As mentioned earlier [1], I'll use python + numpy + matplotlib + some 
netCDF interface (we should be able to support both the google and 
scientific python). I'll start with the basic I/O functionality. I think 
that was quite useful to have some abstraction layer which handles 
common CF netcdf file functionality. I want to then get the EISMINT 
plotting to work again.

I suggest to call the python module PyGC for python glimmer-cism. I'll 
start implementation in the gc2-extensions branch. Please feel free to 
contribute...
